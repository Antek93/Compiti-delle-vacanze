Vue js è un frameworjk per migliorare alcune aree di interazione con il DOM

Un framework = gruppo di strumenti/tools per creare applicazioni web

(è un opensource, leggibile da tutti e modificabile)

crea: interfacce utenti e single app.

Single-Page application: può essere usata su una singola pagina web, cioè il codice è recuperato in un singolo caricamento della pagina

altri framework richiesto dal mondo del lavoro: REACT - ANGULAR

Le API = Application Programming Interface

- Options API
- Composition API

Sono due modi di utilizzare le librerie di VUE

- options EASY
- composition ADVANCED
